# Multi-Agent AI System for Financial Automation

## Overview
A multi-agent AI system designed to automate invoice processing and expense categorization using Python and LLM-based agents.

## Tech Stack
- Python
- Large Language Models (LLMs)
- Agent-based architecture
- REST APIs

## Architecture
- Invoice Processing Agent
- Validation Agent
- Categorization Agent
- Orchestrator Agent

## Results
- ~95% accuracy in invoice classification
- Reduced manual accounting effort by ~60%

## Notes
API keys and confidential data are excluded from this repository.

## Future Scope
- OCR-based document ingestion
- Enhanced inter-agent collaboration
